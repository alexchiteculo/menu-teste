Fortuna work

Framework PhP acelerador lev para Desenvolvimento de web app

O Fortunawork é framework php criado com intento de acelerar a criação de web app, seja site, blog, sistemas de gestão ou qualquer app. Baseado na arquitetura MVC e adoptando os princípios de Progamação Orientada a Objecto o Fortuna work, estrutura toda aplicação e faz com o desenvolvedor se focalize na regra de negócio.

  • MVC
